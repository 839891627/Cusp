# UI Terminology (EN / zh-Hans)

This glossary keeps UI terms consistent across screens and the menu bar.

| English | 简体中文 | Notes |
| --- | --- | --- |
| Overview | 总览 | Primary status dashboard |
| Routes | 连接路由 | Process and route traffic view |
| Strategy | 策略 | Strategy group and node selection area |
| Nodes | 节点 | Individual proxy node entries |
| Subscriptions | 订阅 | Remote source management |
| Logs | 日志 | Runtime and action event history |
| Settings | 设置 | Local app behavior and integration options |
| Runtime Status | 运行状态 | Current runtime health and readiness |
| Readiness | 就绪状态 | Whether startup requirements are satisfied |
| Subscription Refresh | 订阅刷新 | Manual or scheduled source refresh |
| Refresh Interval | 刷新周期 | Refresh cadence selector |
| Run Speed Test | 运行测速 | Probe node latency in current view |
| System Integration | 系统集成 | Startup, restore, notifications, VPN conflict handling |
| Launch at Login | 开机自启 | Start app on macOS login |
| Restore Connection on Launch | 启动时恢复连接 | Reconnect automatically when app launches |
| Auto-Disconnect When Other VPN Is Active | 检测到其他 VPN 时自动断开 | Prevent double-VPN conflicts |
| Recovery & Troubleshooting | 恢复与排障 | Quick recovery actions and install help |
| Restore System Proxy | 恢复系统代理 | Clear residual proxy settings |
| Copy System Commands | 复制系统命令 | Copy `networksetup` fallback commands |
| Open Install Guide | 打开安装指南 | Open unsigned install troubleshooting guide |
| Clear Logs | 清空日志 | Remove current in-app log history |
| Rules | 规则 | Routing rule editor and ordered rule list |
| Import Rules | 导入规则 | Import rules from file |
| Export YAML / Export CONF | 导出 YAML / 导出 CONF | Export current rules in selected format |
| Reset to Preset | 重置为预设 | Restore default rule preset |
| Enabled / Disabled | 已启用 / 已停用 | Boolean source state labels |
